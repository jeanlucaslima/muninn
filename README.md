# Muninn

A macOS menu bar app that records your clipboard history and makes it searchable, recallable, and reusable. Local-first and keyboard-driven.

Named after Muninn, Odin's raven that brings back everything it has seen.

---

## Installation

1. Download `Muninn.app`
2. Drag to Applications
3. Launch the app

---

## First run

When you first launch Muninn:

- It appears as an icon in your menu bar
- The background helper starts automatically and begins recording clipboard history
- You are prompted to optionally:
  - Install the `muninn` CLI (via symlink to your PATH)
  - Enable launch at login

No manual setup required. Everything runs from the app.

---

## Usage

### Menu bar

The menu bar panel is the primary interface.

- `Cmd + Option + V` to open
- Type to search
- Arrow keys to navigate
- `Enter` to copy the selected entry
- `Escape` to close

Click the menu bar icon to toggle the panel.

---

### CLI

The CLI is bundled inside the app. An optional symlink makes the `muninn` command available in your terminal.

```bash
muninn list
muninn search "stripe webhook"
muninn get <id>
muninn copy <id>
```

All commands support `--json` for structured output.

---

## Architecture

```text
Clipboard → Helper → SQLite
                ↓
     ┌──────────┴──────────┐
     │                     │
  Menu Bar              CLI
```

- **Menu bar app** — UI client, the primary interface
- **Helper** — background memory engine (clipboard monitoring, storage, search)
- **CLI** — optional client for scripting and agents

All components communicate through local IPC. No network required.

---

## Data and privacy

- All data is stored locally in Application Support
- No cloud sync
- No external data sharing
- No analytics or telemetry
